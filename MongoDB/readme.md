# MongoDB Project 2

## Description
This project demonstrates basic MongoDB operations using Node.js and the MongoDB Driver.

## Features
- Connect to MongoDB
- Create Database
- Create Collections
- Insert Documents
- Display Documents

## Collections
- Employee
- Inventory
- Scores
- Prices

## Technologies Used
- Node.js
- MongoDB
- MongoDB Node.js Driver
